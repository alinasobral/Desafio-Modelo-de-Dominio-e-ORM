# Desafio - Modelo de Domínio e ORM
Projeto desenvolvido no curso do prof. Nélio Alves utilizando Spring Boot, Hibernate e Banco H2.

## 📚 Tecnologias
-  Java
- Spring Boot
- Maven
- Hibernate
- Banco de dados H2

## ▶️ Como Executar
Execute o comando abaixo na raiz do projeto:
```bash
./mvnw spring-boot:run
```
## 🎯 Objetivo
Aplicar conceitos de:
- Modelagem de domínio
- ORM
